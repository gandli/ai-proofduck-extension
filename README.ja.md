## 概要
AI proofduck はブラウザのサイドパネルで動作するハイブリッド AI ライティング拡張です。

- Chrome Built-in AI (Gemini Nano)
- Local WebGPU/WASM
- Online API (OpenAI-compatible)
- 翻訳フォールバック（無料プロバイダ）
