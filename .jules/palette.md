## 2024-06-26 - Keyboard shortcut hints for submission
**Learning:** Adding visible keyboard shortcut hints (like "⌘ + Enter" for submission) improves usability for power users, but they need to account for both macOS (Cmd) and Windows/Linux (Ctrl) to be universally accessible.
**Action:** When adding shortcut hints in cross-platform tools like browser extensions, dynamically render the modifier key (Cmd/Ctrl) based on the user's OS to prevent confusion.
