## 개요
AI proofduck는 브라우저 사이드패널에서 동작하는 하이브리드 AI 글쓰기 확장입니다.

- Chrome Built-in AI (Gemini Nano)
- Local WebGPU/WASM
- Online API (OpenAI-compatible)
- 번역 폴백(무료 제공자)
