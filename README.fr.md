## Aperçu
AI proofduck est une extension d’assistant d’écriture IA hybride pour le panneau latéral du navigateur.

- Chrome Built-in AI (Gemini Nano)
- Local WebGPU/WASM
- Online API (OpenAI-compatible)
- Fallback de traduction (services gratuits)
