## 2025-02-24 - [Replace predictable Math.random() with Web Crypto API for secure GUID generation]
**Vulnerability:** Predictable PRNG (`Math.random()`) was used for generating GUIDs in `src/services/SpeechService.ts`.
**Learning:** Hardcoded, non-cryptographic fallbacks are common in legacy code or components expecting varying environments. This poses a risk if these identifiers are used for session tracking, message routing, or any security-sensitive context. While the fallback to `Math.random()` might be necessary for compatibility, prioritizing `crypto.randomUUID()` and `crypto.getRandomValues()` significantly improves the security posture for the vast majority of modern clients.
**Prevention:** Always default to using the Web Crypto API (`crypto.randomUUID()` or `crypto.getRandomValues()`) for generating any random identifiers or tokens. Only use `Math.random()` as a last resort in non-critical environments, and ensure appropriate feature detection is in place.
