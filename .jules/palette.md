## 2025-02-20 - Tailwind CSS focus-visible for Keyboard Accessibility
**Learning:** For interactive UI elements (like custom buttons and tabs) built with Tailwind CSS, using standard `focus-visible:outline-none` combined with `focus-visible:ring-2 focus-visible:ring-brand-orange/50` provides a highly visible and accessible focus indicator for keyboard users while maintaining visual design consistency. `focus-visible:ring-offset-2` should be used for solid buttons to prevent the ring from blending in, and `focus-visible:ring-inset` should be used for tabs to avoid visual clipping.
**Action:** When implementing new interactive components or fixing existing ones in this codebase, ensure keyboard accessibility is addressed using these standardized Tailwind `focus-visible` classes.
