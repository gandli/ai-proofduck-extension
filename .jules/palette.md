## 2024-06-01 - Keyboard Accessibility Focus States
**Learning:** Tailwind's default focus styles can be hard to see against certain backgrounds. For high-contrast interfaces, combining `focus-visible:outline-none` with explicit colored rings (`focus-visible:ring-2 focus-visible:ring-brand-orange/50`) works best. For elements with identical background colors to the ring, using `focus-visible:ring-offset-2` (for buttons) or `focus-visible:ring-inset` (for tabs to avoid clipping borders) is crucial to ensure the focus indicator remains visible.
**Action:** When styling interactive elements for keyboard accessibility (tabs, buttons, switches), consistently apply `focus-visible` styles with appropriate ring colors and offsets tailored to the element's layout context to guarantee visibility.
