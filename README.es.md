## Resumen
AI proofduck es una extensión de escritura con IA híbrida para el panel lateral del navegador.

- Chrome Built-in AI (Gemini Nano)
- Local WebGPU/WASM
- Online API (OpenAI-compatible)
- Respaldo de traducción (proveedores gratuitos)
