# Research Summary

## Stack

项目继续采用 WXT + React + TypeScript + Tailwind，测试使用 Vitest + Playwright。

## Table Stakes

v1 至少需要：侧边栏、选中文本处理、整页抓取、翻译、摘要、校对、润色、扩写、结果展示、设置与引擎切换。

## Differentiators

真正的差异化不在“模式数量”，而在“本地优先 + 多引擎 + 低打断网页流程”。

## Watch Out For

- 不要把它做成泛聊天产品。
- 不要在 v1 引入账号和云平台。
- 不要让本地模型加载体验失控。
